# Koa 2 + Passport + Mongoose + GraphQL

## Command

#### Setup
```bash
npm install
```
#### Develop
```bash
# using nodemon
npm start
```

## Dependencies

- Watcher and hot-reload: [nodemon](http://nodemon.io/)
